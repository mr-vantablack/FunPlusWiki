
# Welcome to Fun Plus Essentials mod wiki!

FunPlusEssentials is a mod that adds new features and bug fixes to the game. The mod was written by **Vantablack**.

This mod adds a lot of features to the game:
- Custom maps
- Music player
- Useful tweaks
- Bug fixes
- Funny features
